STEP 1 — ONCE PER YEAR

Season Setup

Create the season before the first round. The system handles the rest.

Create the Season

Admin → Season → Create Season

1 Set the season name (e.g., "2026 Wolf Cup Season")
2 Set start date (first Friday) and end date (last Friday)
3 The system auto-creates a round for every Friday with rotating tees: Black → Blue → White → repeat
4 Uncheck any weeks that won't be played (e.g., Member-Guest). Change tees for any specific week.
Enable "Harvey Live" to show projected Harvey Cup points on the leaderboard during rounds.

STEP 2 — MANAGE YOUR PLAYERS

Roster Management

Add players, pull handicaps from GHIN, and set who's active for the season.

The Roster

Admin → Roster

1 Add new players — enter their name and GHIN number
2 Tap the status badge to cycle: Active → Sub → Inactive. Active and Sub players appear in attendance.
3 Tap Refresh All Handicaps to pull the latest GHIN indexes for everyone at once.
Each player's handicap is snapshotted when the round is created — a mid-round GHIN change won't affect an active round.

EVERY FRIDAY

Taking Attendance

The Attend tab shows this week's round. Mark who's playing, then create the round.

Who's Playing?

Attend tab (bottom nav)

1 Tap each player to toggle in (green dot) or out (red dot). Work through the list from the GroupMe chat.
2 The confirmed count shows at the top (e.g., "8/19 confirmed"). Must be a multiple of 4.
3 Need a sub? Scroll down and tap Add Sub — search by name or GHIN number.
4 Scroll to the bottom and tap Create Round. The join code is the year: 2026.
The week, date, and tee color are shown at the top. The system automatically shows the next upcoming Friday.

GAME DAY

Groups & Pairings

Use auto-suggest to minimize repeat pairings, or build groups manually.

Suggest & Apply

Pairings Sheet

1 Check off who's playing in the Who's Playing checklist
2 Tap Suggest — the algorithm minimizes repeat pairings from the season
3 Tap Apply to lock it in, or Re-roll for a different draw
4 Tap Pairings to see the shareable group sheet with course handicaps

DURING THE ROUND

Scoring & Finalization

Players enter scores on their phones. You finalize when everyone's done.

Players Join & Score

Tell the group: go to wolf.dagle.cloud → Score → enter code 2026

1 One person per group runs scoring. They do the ball draw and enter scores hole by hole.
2 The leaderboard updates live as scores come in. Anyone can watch — no login needed.
3 When all groups finish hole 18, go to Admin → Rounds and tap Finalize.
Finalization locks scores and computes Harvey Cup points. You can still make corrections afterwards.

The Leaderboard

Shows all players ranked by Harvey points with stableford, money, and to-par. Tap any player to expand their full scorecard with wolf decisions on every hole.

AFTER THE ROUND

Score Corrections

Fix mistakes with a full audit trail. The system recalculates everything automatically.

Making a Correction

Admin → Score Corrections

1 Select the finalized round from the dropdown
2 Pick the hole number and player
3 Enter the correct gross score
4 Tap Submit Correction — money, stableford, and Harvey points recalculate for the entire group
Every correction is logged in the Audit Log with timestamp, admin name, old score, new score. Nothing is hidden.

TROUBLESHOOTING

Player can't join?

Make sure the round is active (not just scheduled). Confirm they're using code 2026. Try a hard refresh.

Wrong score entered?

During the round: go back to the hole and change it. After finalization: use Score Corrections in admin.

Handicaps look wrong?

Tap Refresh All Handicaps on the Roster page. Handicaps are snapshotted at round creation.

Need to cancel a round?

Admin → Rounds → Cancel. Cancelled rounds don't count for standings or Harvey Cup.
